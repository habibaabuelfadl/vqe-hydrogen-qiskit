# Variational Quantum Eigensolver (VQE) using Qiskit

This project implements the Variational Quantum Eigensolver (VQE) algorithm using Qiskit to estimate the ground-state energy of a molecular system.

## Features
- Built parameterized quantum circuits (ansatz)
- Used classical optimization to minimize energy
- Simulated quantum measurements using Qiskit
- Compared VQE results with exact eigensolver solutions
- Visualized optimization convergence

## Tools
- Python
- Qiskit
- NumPy
- Matplotlib

## Results
The VQE workflow demonstrates hybrid quantum-classical optimization for estimating molecular ground-state energies.
